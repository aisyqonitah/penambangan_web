# Kesimpulan dan Referensi

### **Kesimpulan**

Setelah mengambil 153 data pada website <https://palominobag.com/best-seller> kita mendapatkan data-data yang dapat dihitung jumlah katanya, dinilai tingkat kemiripan setiap datanya, dan dimasukkan dalam cluster yang sama jika tingkat kemiripannya sama. Disini vsm, tfidf, seleksi fitur, dan clustering memiliki pengaruh untuk menentukan itu semua.
